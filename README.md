
# Pathfinder Visualization Tool

## 📌 Description

This project is a **pathfinding visualization tool** that demonstrates how the **A* (A-star) algorithm** finds the shortest path between two points.

The application allows users to interactively create a grid, generate mazes, and visualize how the algorithm explores the grid step by step.

## 🚀 Features

* Visual representation of the **A*** pathfinding algorithm
* Interactive grid system
* Wall/obstacle drawing
* Two different **maze generation algorithms**
* Real-time pathfinding animation

## 🧠 Algorithms Used

* **A*** (A-star) for shortest path calculation
* Maze generation algorithms:

  * Recursive Division (example)
  * Depth first search (example)
